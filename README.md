# Raylib Examples

I'm making this translation of raylib examples to f# as a way to learn the language.

### Dependency

-[.Net 8](https://dotnet.microsoft.com/pt-br/download/dotnet/8.0)

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Run

```bash
  dotnet run
```
